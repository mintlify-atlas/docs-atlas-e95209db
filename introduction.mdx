## What is Frappe Wiki?

Frappe Wiki is an open source wiki and knowledge base platform designed to serve dynamic, text-heavy content like documentation and internal knowledge bases. Built on the powerful Frappe Framework, it provides a delightful experience for both writers and readers.

The platform was developed by Frappe to meet their own documentation needs. When they couldn't find a simple, clean, open source wiki for ERPNext documentation, they built Frappe Wiki from scratch. Today, it powers their user manuals, company policies, and more across their entire organization.

### Architecture Highlights

- **Backend**: Python-based Frappe app with DocTypes for Wiki Documents, Spaces, and Change Requests
- **Frontend**: Vue 3 SPA with TipTap editor for rich markdown editing
- **Search**: RedisSearch or SQLite-based full-text search
- **Tree Structure**: Nested set model for hierarchical wiki organization
- **Change Management**: Git-inspired revision and merge system for collaborative editing

## Use Cases

Frappe Wiki excels at:

- **Product Documentation**: Create comprehensive user guides and API references
- **Internal Knowledge Bases**: Centralize company policies, procedures, and best practices
- **Team Wikis**: Collaborate on shared knowledge with approval workflows
- **Public Documentation Sites**: Host open source project documentation with custom branding

<Note>
  Frappe Wiki maintains revision history and has a change approval mechanism, making it perfect for teams that need controlled content publishing.
</Note>
