The Frappe Wiki frontend is a modern single-page application built with Vue 3, providing an interactive interface for document editing and wiki management.

## Technology Stack

### Core Framework

- **Vue 3** - Progressive JavaScript framework with Composition API
- **Vue Router** - Client-side routing
- **Pinia** - State management
- **Vite** - Build tool and dev server

### UI Layer

- **Frappe UI** - Component library and utilities
  - Pre-built components (Button, Input, Dialog, etc.)
  - Resource management (`resourcesPlugin`)
  - Request handling (`frappeRequest`)
  - Page metadata (`pageMetaPlugin`)

### Rich Text Editing

- **TipTap** - ProseMirror-based rich text editor
- Custom extensions for Wiki-specific functionality

## Application Bootstrap

### Entry Point (`main.js`)

```javascript
import { createApp } from 'vue';
import App from './App.vue';
import router from './router';
import { initSocket } from './socket';
import { pinia } from './stores';
import translationPlugin from './translation';
import {
  resourcesPlugin,
  pageMetaPlugin,
  setConfig,
  frappeRequest
} from 'frappe-ui';

const app = createApp(App);

// Configure Frappe UI resource fetcher
setConfig('resourceFetcher', frappeRequest);

// Install plugins
app.use(pinia);
app.use(router);
app.use(translationPlugin);
app.use(resourcesPlugin);
app.use(pageMetaPlugin);

// Initialize WebSocket
const socket = initSocket();
app.config.globalProperties.$socket = socket;

// Register global components
app.component('Button', Button);
app.component('TextInput', TextInput);
// ... more global components

app.mount('#app');
```

### Root Component (`App.vue`)

```vue
<script setup>
import { FrappeUIProvider, setConfig } from "frappe-ui";
import MainLayout from "./layouts/MainLayout.vue";

setConfig("systemTimezone", window.timezone?.system || null);
setConfig("localTimezone", window.timezone?.user || null);
</script>

<template>
  <FrappeUIProvider>
    <MainLayout>
      <router-view />
    </MainLayout>
  </FrappeUIProvider>
</template>
```

**Key Features:**
- `FrappeUIProvider` wraps the entire app for Frappe UI context
- `MainLayout` provides consistent layout structure
- `router-view` renders current route component

## Routing Architecture

### Route Configuration

```javascript
const routes = [
  {
    path: '/',
    redirect: '/spaces',
  },
  {
    path: '/spaces',
    name: 'SpaceList',
    component: () => import('@/pages/Spaces.vue'),
  },
  {
    path: '/spaces/:spaceId',
    component: () => import('@/pages/SpaceDetails.vue'),
    props: true,
    children: [
      {
        path: '',
        name: 'SpaceDetails',
        component: () => import('@/components/SpaceWelcome.vue'),
      },
      {
        path: 'page/:pageId',
        name: 'SpacePage',
        component: () => import('@/components/WikiDocumentPanel.vue'),
        props: true,
      },
      {
        path: 'draft/:docKey',
        name: 'DraftChangeRequest',
        component: () => import('@/components/DraftContributionPanel.vue'),
        props: true,
      },
    ],
  },
  {
    path: '/change-requests',
    name: 'ChangeRequests',
    component: () => import('@/pages/Contributions.vue'),
  },
  {
    path: '/change-requests/:changeRequestId',
    name: 'ChangeRequestReview',
    component: () => import('@/pages/ContributionReview.vue'),
    props: true,
  },
];

const router = createRouter({
  history: createWebHistory('/wiki'),
  routes,
});
```

**Base Path:** All routes are under `/wiki` path.

### Authentication Guard

```javascript
router.beforeEach(async (to, from, next) => {
  const sessionStore = useSessionStore();
  const userStore = useUserStore();
  
  let isLoggedIn = sessionStore.isLoggedIn;
  
  try {
    if (!userStore.data) {
      await userStore.fetch();
    }
  } catch (error) {
    isLoggedIn = false;
  }
  
  if (!isLoggedIn) {
    window.location.href = `/login?redirect-to=/wiki/${encodeURIComponent(to.fullPath)}`;
  } else {
    next();
  }
});
```

## State Management (Pinia)

### Session Store (`stores/session.js`)

**Purpose:** Manage user authentication state.

**State:**
- `isLoggedIn` - Authentication status
- `user` - Current user information

**Actions:**
- `checkAuth()` - Verify authentication
- `logout()` - End session

### User Store (`stores/user.js`)

**Purpose:** Manage user profile data.

**State:**
- `data` - User profile object

**Actions:**
- `fetch()` - Load user data from API

### Change Request Store (`stores/changeRequest.js`)

**Purpose:** Manage active change request state.

**State:**
- `currentRequest` - Active change request data
- `conflicts` - Merge conflicts
- `reviewers` - Assigned reviewers

**Actions:**
- `load(id)` - Load change request
- `save()` - Save changes
- `submit()` - Submit for review
- `approve()` - Approve changes
- `merge()` - Merge to main

**Purpose:** Format change types for display.

**Purpose:** Handle sidebar resize interactions.

## Data Fetching with Frappe UI

### Resource Pattern

Frappe UI provides a `createResource` utility for API interactions:

```javascript
import { createResource } from 'frappe-ui';

const wikiSpaces = createResource({
  url: 'wiki.api.get_spaces',
  auto: true, // Fetch on mount
  transform: (data) => {
    // Transform response data
    return data.map(space => ({
      ...space,
      route: `/wiki/spaces/${space.name}`
    }));
  },
});

// In component
watch(() => wikiSpaces.data, (spaces) => {
  console.log('Spaces loaded:', spaces);
});

// Manual fetch
await wikiSpaces.fetch();

// Reload
await wikiSpaces.reload();
```

**Features:**
- Automatic loading states
- Error handling
- Caching
- Request deduplication
- Transform pipeline

## Real-Time Features

### Socket.io Integration

**Purpose:** Real-time updates for collaborative editing.

**Events:**
- `doc_update` - Document changed by another user
- `doc_delete` - Document deleted
- `doc_move` - Document moved in tree
- `change_request_update` - Change request status changed

**Usage:**
```javascript
const socket = inject('$socket');

socket.on('doc_update', (data) => {
  // Refresh document view
  reloadDocument(data.doc_key);
});
```

**Boot Context:** Provided by `wiki/www/wiki.py`

```python
def get_boot():
    return frappe._dict({
        "frappe_version": frappe.__version__,
        "site_name": frappe.local.site,
        "read_only_mode": frappe.flags.read_only,
        "system_timezone": get_system_timezone(),
    })
```
