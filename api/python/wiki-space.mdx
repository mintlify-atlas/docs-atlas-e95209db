### update_routes(new_route)

Updates the Wiki Space route and all child Wiki Documents.

```python
@frappe.whitelist()
def update_routes(self, new_route: str) -> dict
```

<ParamField path="new_route" type="str" required>
  The new route for the wiki space
</ParamField>

**Returns:** `dict` - Dictionary with `updated_count` key

**Raises:** 
- `frappe.PermissionError` - If user is not Wiki Manager
- `frappe.ValidationError` - If route is invalid or already exists

**Example:**
```python
space = frappe.get_doc("Wiki Space", "SPACE-001")
result = space.update_routes("new-docs")
print(f"Updated {result['updated_count']} documents")
```

## Usage Examples

### Creating a New Wiki Space

```python
import frappe

# Create a new wiki space
space = frappe.new_doc("Wiki Space")
space.space_name = "Product Documentation"
space.route = "product-docs"
space.is_published = 1
space.enable_feedback_collection = 1
space.insert()

# The root group is automatically created
print(f"Root group: {space.root_group}")
```

### Adding Documents to a Space

```python
import frappe

space = frappe.get_doc("Wiki Space", "SPACE-001")

# Create a new document under the space
doc = frappe.new_doc("Wiki Document")
doc.title = "Getting Started"
doc.parent_wiki_document = space.root_group
doc.content = "# Welcome\n\nStart here..."
doc.is_published = 1
doc.insert()
```

### Updating Space Routes

```python
import frappe

space = frappe.get_doc("Wiki Space", "SPACE-001")

# Update the route (updates all child documents)
result = space.update_routes("new-documentation")
print(f"Updated {result['updated_count']} documents")

# All document routes are automatically updated
```

### Cloning a Wiki Space

```python
import frappe

space = frappe.get_doc("Wiki Space", "SPACE-001")

# Clone the space asynchronously
result = space.clone_wiki_space_in_background("docs-v2")

if result['status'] == 'queued':
    print(f"Cloning to route: {result['new_route']}")
    print("Clone will complete in the background")
```
