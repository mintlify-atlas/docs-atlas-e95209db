The `WikiChangeRequest` class represents a change request (CR) for wiki documents. It provides a Git-like workflow for proposing, reviewing, and merging changes.

## Class Definition

```python
from frappe.model.document import Document

class WikiChangeRequest(Document):
    pass
```

**Location:** `wiki/frappe_wiki/doctype/wiki_change_request/wiki_change_request.py`

### create_cr_page(name, parent_key, title, ...)

Creates a new page in a change request.

```python
@frappe.whitelist()
def create_cr_page(
    name: str,
    parent_key: str,
    title: str,
    slug: str | None = None,
    is_group: int = 0,
    is_published: int = 1,
    content: str | None = None,
    order_index: int | None = None,
    is_external_link: int = 0,
    external_url: str | None = None,
) -> str
```

<ParamField path="name" type="str" required>
  Name of the change request
</ParamField>

<ParamField path="parent_key" type="str" required>
  The doc_key of the parent document
</ParamField>

<ParamField path="title" type="str" required>
  Title of the new page
</ParamField>

<ParamField path="slug" type="str">
  URL slug (auto-generated if not provided)
</ParamField>

<ParamField path="is_group" type="int" default="0">
  Whether this is a group/folder (1) or page (0)
</ParamField>

<ParamField path="is_published" type="int" default="1">
  Whether the page is published
</ParamField>

<ParamField path="content" type="str">
  Markdown content of the page
</ParamField>

<ParamField path="order_index" type="int">
  Order among siblings (auto-incremented if not provided)
</ParamField>

<ParamField path="is_external_link" type="int" default="0">
  Whether this links to an external URL
</ParamField>

<ParamField path="external_url" type="str">
  External URL if is_external_link is 1
</ParamField>

**Returns:** `str` - The doc_key of the newly created page

**Example:**
```python
import frappe

new_doc_key = frappe.call(
    "wiki.frappe_wiki.doctype.wiki_change_request.wiki_change_request.create_cr_page",
    name="CR-001",
    parent_key="doc-key-parent",
    title="New Feature Guide",
    content="# New Feature\n\nThis guide covers...",
    is_published=1
)
```
