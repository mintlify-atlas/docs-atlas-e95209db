The `WikiDocument` class represents a document in the Frappe Wiki system. It extends `NestedSet` to provide hierarchical document organization.

## Class Definition

```python
from frappe.utils.nestedset import NestedSet

class WikiDocument(NestedSet):
    pass
```

**Location:** `wiki/frappe_wiki/doctype/wiki_document/wiki_document.py`
