The `WikiRevision` class represents a snapshot of wiki documents at a point in time. It provides version control functionality similar to Git commits.

## Class Definition

```python
from frappe.model.document import Document

class WikiRevision(Document):
    pass
```

### ensure_overlay_item(revision, doc_key)

Copy-on-write: ensures a doc_key has an item in the overlay revision.

```python
def ensure_overlay_item(revision: str, doc_key: str) -> str | None
```

<ParamField path="revision" type="str" required>
  Name of the overlay revision
</ParamField>

<ParamField path="doc_key" type="str" required>
  The doc_key to ensure exists in overlay
</ParamField>

**Returns:** `str | None` - Name of the Wiki Revision Item, or None if item doesn't exist

**Description:**
If the item already exists in the overlay, returns its name.
If the revision is an overlay and the item exists in the base, copies it to the overlay.

**Example:**
```python
from wiki.frappe_wiki.doctype.wiki_revision.wiki_revision import ensure_overlay_item

item_name = ensure_overlay_item("REV-OVERLAY-001", "doc-key-123")
if item_name:
    # Item now exists in the overlay and can be modified
    item = frappe.get_doc("Wiki Revision Item", item_name)
    item.title = "Updated Title"
    item.save()
```

## Usage Examples

### Creating a Revision from Live Documents

```python
import frappe
from wiki.frappe_wiki.doctype.wiki_revision.wiki_revision import create_revision_from_live_tree

# Snapshot the current state
revision = create_revision_from_live_tree(
    wiki_space="SPACE-001",
    message="Documentation update - Q1 2024"
)

print(f"Created revision {revision.name}")
print(f"Tree hash: {revision.tree_hash}")
print(f"Document count: {revision.doc_count}")
```

### Working with Overlay Revisions

```python
import frappe
from wiki.frappe_wiki.doctype.wiki_revision.wiki_revision import (
    create_overlay_revision,
    ensure_overlay_item,
    mark_hashes_stale
)

# Create an overlay for making changes
overlay = create_overlay_revision(
    base_revision="REV-001",
    is_working=1
)

# Copy-on-write: modify an item
item_name = ensure_overlay_item(overlay.name, "doc-key-123")
if item_name:
    item = frappe.get_doc("Wiki Revision Item", item_name)
    item.title = "Updated Title"
    item.save()
    
    # Mark hashes as stale after changes
    mark_hashes_stale(overlay.name)
```

### Comparing Revisions

```python
import frappe
from wiki.frappe_wiki.doctype.wiki_revision.wiki_revision import (
    get_revision_item_map,
    ensure_revision_hashes
)

# Ensure hashes are up-to-date
ensure_revision_hashes("REV-001")
ensure_revision_hashes("REV-002")

# Get revision info
rev1 = frappe.get_doc("Wiki Revision", "REV-001")
rev2 = frappe.get_doc("Wiki Revision", "REV-002")

if rev1.tree_hash != rev2.tree_hash:
    print("Tree structure has changed")

if rev1.content_hash != rev2.content_hash:
    print("Content has changed")
```

### Building Ordered Document List

```python
import frappe
from wiki.frappe_wiki.doctype.wiki_revision.wiki_revision import (
    get_revision_item_map,
    build_tree_order
)

item_map = get_revision_item_map("REV-001")
ordered_keys = build_tree_order(item_map)

# Process documents in tree order
for doc_key in ordered_keys:
    item = item_map[doc_key]
    if not item['is_deleted']:
        print(f"{'  ' * item.get('depth', 0)}{item['title']}")
```
